HTML STRUCTURE

2.1. Basic Structure

The Basic Structure contains:

<div class="site">

<header class="noo-header header-1"></header> - header content

<div class=" main"></div> - all the content

<footer class=" wrap-footer footer-2 colophon wigetized"></footer> - footer content

</div>

SITE LAYOUT

5.1. Wide Layout

Wide layout is default layout.

 

5.2. Boxed Layout

You need to add class boxed-layoutin each body tag.

HTML

<body class="boxed-layout">

{{BODY CONTENT}}

</body>

 

CSS – STYLE.CSS

.boxed-layout {
    background-color: #ffffff;
    background-image: url("");
    background-position: left top;
    background-repeat: no-repeat;
    background-attachment: fixed;
}

.boxed-layout .site {
    margin: 0 auto;
    position: relative;
    padding: 0;
    background-color: #ffffff;
    -webkit-box-shadow: 0 0 7px rgba(0, 0, 0, 0.4);
    box-shadow: 0 0 7px rgba(0, 0, 0, 0.4);
    width: 90%;
    max-width: 1400px;
	overflow-x: hidden;
}

.boxed-layout .site .fixed_top .navbar-wrapper {
    width: 90%;
    max-width: 1200px;
    margin: 0 auto;
}

.boxed-layout .site .header-1 .noo-menu-option a {
	font-size: 0;
}

.boxed-layout .site .header-1 .noo-menu-option a i {
	font-size: 14px;
}

}

COLORS

CSS for skin color of the theme is located in folder “/css/color”:

  1. blue.css
  2. cyan.css
  3. green.css
  4. pink.css
  5. red.css
  6. yellow.css

HTML

<link rel="stylesheet" href="css/colors/cyan.css">

 

To select other colors, insert file blue.css, cyan.css, green.css, pink.css, red.css or yellow.css corresponding to the color you want.
